# trabajo-California-Traffic-Collision.
Este proyecto realiza un análisis exploratorio de los accidentes de tráfico en California utilizando datos públicos del sistema SWITRS (Statewide Integrated Traffic Records System).

El objetivo del proyecto es analizar patrones, tendencias y factores asociados a los accidentes de tráfico, utilizando herramientas de análisis de datos, visualización y desarrollo de aplicaciones.

El proyecto incluye:
-Limpieza y transformación de datos
-Análisis exploratorio
-Visualización de resultados
-Un dashboard interactivo
-Una aplicación desarrollada con Streamlit

-----------------------------------------------------------------------------------
## Dataset
Los datos originales provienen del dataset público de Kaggle:

California Traffic Collision Data from SWITRS

Este dataset contiene información sobre colisiones de tráfico reportadas por la policía en California, incluyendo variables como:

-Fecha y hora del accidente
-Ubicación
-Tipo de colisión
-Condiciones de la carretera
-Factores asociados (alcohol, velocidad, etc.)
-Información sobre víctimas y vehículos

Los datos provienen del sistema SWITRS, mantenido por la California Highway Patrol, que recopila los accidentes reportados por las autoridades en todo el estado.

## Modificación del dataset

Para este proyecto:

-El dataset original fue filtrado y limpiado.
-Se filtraron los registros para su uso, desde el año 2018 al 2021.
-Se seleccionaron únicamente las variables relevantes para el análisis.

La base de datos modificada se encuentra en el repositorio dentro de un archivo .zip.
